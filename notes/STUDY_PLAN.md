# Study Plan - Recommended Order

## 🎯 Recommended Learning Path

### Month 1-2: Foundations + Practice

**Week 1-2: Systems Monitor (Start Easy)**
- Introduces FFI, cross-platform code
- Less complex than others
- Builds confidence
- **Book chapters:** 6, 9, 11, 12

**Week 3-4: Distributed Cache**
- Introduces unsafe code, concurrency
- Practical application
- Real performance challenges
- **Book chapters:** 1-4, 9-11

### Month 3-4: Advanced Patterns

**Week 5-6: Database Engine**
- Complex data structures
- Transaction logic
- Type-safe APIs
- **Book chapters:** 1-6, 9-11

**Week 7-8: Async Runtime**
- Deepest understanding of Rust internals
- Pin, Future, lifetimes
- Hardest project
- **Book chapters:** 1-3, 5, 7-8, 12

### Month 5-6: Mastery

**Week 9-12: VM Runtime**
- Combines everything
- Your masterpiece
- Most creative freedom
- **Book chapters:** ALL
